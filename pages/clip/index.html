Используем Clip для ваших элементов в HTML

Первый случай моего использования я уже не найду, однако, из последнего вы сможете наблюдать на этом сайте: https://tipo.intrid.ru/

В данном случае я использовал Clip для создания интерактивной карточки услуги для клиентов. Border как бы проходит по касательной всех изгибов и в результате карточка имеет форму, а не просто прямоугольник.

Чтобы использовать Clip, вы можете просто добавить атрибут `clip-path`. Однако, чтобы это сработало так, как показано на скрине - потребуется добавить дополнительную обертку, которая в свою очередь будет иметь схожий `clip-path`.
